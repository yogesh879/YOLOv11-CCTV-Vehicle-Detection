## Objective
To implement YOLOv11 for real-time object detection of cars and bikes using CCTV footage.

## Features
- Detect cars and bikes
- Real-time video processing
- High accuracy detection

## Technologies

- OpenCV
- YOLOv11
- Google Colab

## How to Run
1. Open in Google Colab
2. Install libraries
3. Run all cells
4. Upload video
